## Introduction to Application Security
TapTap Developer Services provides an app security module that protects the safety of game apps through APK hardening and anti-cheat detection.

## Rules 

- Applicable games: games that have joined the Fire Plan and are running normally, or games exclusively released on the TapTap platform.
